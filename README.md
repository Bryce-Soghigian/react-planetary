# Key Features

- v1.0 (current)
	- Can Import all Planets into your application
      
- v1.1 (next)
	- Can pass props to all of the planets (orbital_controls=bool, size=num rotation etc etc)
	
- v1.2 (future)
	- Can import all planets into the same canvas and adjust their positions
